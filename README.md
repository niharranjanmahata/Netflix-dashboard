# 🎬 Netflix Dashboard

A Netflix Data Analysis Dashboard built using Python and Streamlit. This project analyzes the Netflix Titles dataset and presents insights through interactive charts and visualizations.

## 📌 Features

- 📊 Movies vs TV Shows Analysis
- 🌍 Country-wise Content Distribution
- ⭐ Rating Distribution

- 🔍 Interactive Dashboard
- 📈 Data Visualization

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📂 Project Structure

```
Netflix Dashboard/
│── app.py
│── visualization.py
│── NetflixMovieSeries.ipynb
│── netflix_titles.csv
│── requirements.txt
```

## 🚀 Installation

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

## 📊 Dataset

Netflix Titles Dataset containing information about Movies and TV Shows available on Netflix.
